About

Expert System is designed as a machine learning playground that allows you to experiment with different datasets and algorithms. You can upload your own dataset or use a preloaded dataset to train and test machine learning models. The application supports classification, regression, and clustering algorithms. You can visualize the results of your models and evaluate their performance using various metrics.

The application is built using Django, a high-level Python web framework, and scikit-learn, a popular machine learning library in Python. The frontend is designed using HTML, CSS, and JavaScript, with Bootstrap for styling and jQuery for interactivity. The application is hosted on Microsoft Azure, a cloud platform as a service, and uses SQLite as the database for temporary storage.

The source code for this project is available on GitHub.
